# Sever Management and Information

## Server

All the information and settings to do with your server, segregated into tabs:

### Statistics

* Data metrics for the server's bandwidth and hardware (temperature and fans)
  * Bandwidth displayed by a graph, or click the "Breakdown" option in the top right.
  * IPMI Sensors displayed by a graph

<figure><img src="../../.gitbook/assets/image (74).png" alt=""><figcaption><p>Bandwidth Traffice Usage Breakdown</p></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (75).png" alt=""><figcaption><p>IPMI Sensors Graphs</p></figcaption></figure>

### IP Management

* All the information surrounding your IP address covered under IP Address Overview
  * The Primary Address, Netmask, Gateway, Description

<figure><img src="../../.gitbook/assets/image (76).png" alt=""><figcaption><p>IP Address Overview Information</p></figcaption></figure>

### Configuration

* The main details of your server
  * Server Name, Hostname, OS, Assigned Client, Server Type, & General Description
    * You cannot touch any of this information, other than setting a "General Description". Be sure to click "Update" after you're done!

<figure><img src="../../.gitbook/assets/image (77).png" alt=""><figcaption><p>Configuration Tab Main Details and General Description</p></figcaption></figure>

### Notes

* Displays all a catalogue of the notes you have created (in the area currently indicating "no notes").
  * You can create notes by writing in the box under "General Description". Be sure to click "Update" (again) after you're done!

<figure><img src="../../.gitbook/assets/image (78).png" alt=""><figcaption><p>Notes Tab Log and Tool</p></figcaption></figure>

### Activity Log

* And finally, the Activity Log tab will display the log of your activities
  * Date, User, Description, & IP Location

<figure><img src="../../.gitbook/assets/image (79).png" alt=""><figcaption><p>Activity Log</p></figcaption></figure>
