# Client Area

## Dedicated Server Dashboard

Welcome to your [Dedicated Server Dashboard](https://dedicated.1of1servers.com/servers)!

The 1Of1 Dedicated Server Dashboard provides you with a broad view of your account's activity and information, it displays:&#x20;

{% tabs %}
{% tab title="Servers" %}
Servers Menu

* All of your servers & their basic information:
  * Power Status (On/Off), Server Name, Hostname, Primary IP Address, Operating System, & the server Type (Dedicated/VPS/Game/etc).
{% endtab %}

{% tab title="My Account" %}
Account Menu

* Account Details&#x20;
  * Client Name, Username, Password, Two Factor Authentication, Email, Date & Time, IP Restrictions, & your assigned servers.
* Login History&#x20;
  * Date & Time, IP-Address, & User-agent.
* Access Log
  * Date & Time, Response Code, Method, Request Source, & HTTP Endpoint.
{% endtab %}

{% tab title="Logs" %}
Logs Menu

* User Log
  * Date & Time, User ID, Result, Log, Affects, User IP, Request Source, & Action.
{% endtab %}
{% endtabs %}

{% hint style="info" %}
First Things First!

* Take care of your eyes and select your favourite theme!
  * Top Right corner, select the dropdown and turn On/Off
* Enable Two Factor Authentication
  * Top Left corner, select "**My Account**", then locate the "**Activate 2FA**" across from the "**Password**" box
{% endhint %}

* The server's specs (CPU/RAM/Storage/Traffic) and the Primary IP Address
* Information & Settings Tabs:
  * Statistics page
  * Configuration page
  * Notes Page
  * Activity Log Page
