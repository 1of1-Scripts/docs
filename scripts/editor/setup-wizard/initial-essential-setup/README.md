# Initial Essential Setup

This section covers the foundation of essential items to be completed for your web queue to be up and running:

* FiveM API Configuration
* Discord Setup
* Tebex API Configuration (coming soon...)

To access the webqueue dashboard, you must:

1. Purchase the webqueue via our Tebex store
2. Join the 1OF1 Scripts Discord
3. Obtain your roles (Customer)
4. Go to [https://waitqueue.com/](https://waitqueue.com/) sign in and link your Discord.
