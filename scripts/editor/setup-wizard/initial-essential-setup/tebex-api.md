# Tebex API

{% stepper %}
{% step %}
### Tebex API Setup Menu

Enter the Tebex API Configuration menu

<figure><img src="../../../../.gitbook/assets/image (49).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
### Add API Key

Click "Setup Configuration"

<figure><img src="../../../../.gitbook/assets/image (50).png" alt=""><figcaption></figcaption></figure>

Then go to [https://creator.tebex.io/game-servers](https://creator.tebex.io/game-servers) and click "Edit" on your game servers Tebex store.

<figure><img src="../../../../.gitbook/assets/image (51).png" alt=""><figcaption></figcaption></figure>

Copy your Secret Key

<figure><img src="../../../../.gitbook/assets/image (52).png" alt=""><figcaption></figcaption></figure>

Then paste it into the Tebex API Key field in the Web Queue Dashboard.

Save your configuration.
{% endstep %}

{% step %}
Tebex Payments

A new section will populate on the side-bar called Integrations, within this category you will find Tebex Payments.

<figure><img src="../../../../.gitbook/assets/image (53).png" alt=""><figcaption></figcaption></figure>

Under Tebex Payment you are able to see the transaction history of your Tebex store.

<figure><img src="../../../../.gitbook/assets/image (54).png" alt=""><figcaption></figcaption></figure>

Click "View" on a transaction to view additional details of the purchase.&#x20;

<figure><img src="../../../../.gitbook/assets/image (56).png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
