# Module Settings

{% stepper %}
{% step %}
## Queue Module

Controls all queue-related functionality.

* **Enable Queue Module**\
  When disabled, all queue features are hidden from players and moderators.
* **Lock Queue**\
  When locked, **new players cannot join the queue**. Players already in the queue can still be process
{% endstep %}

{% step %}
## Applications Module

Controls all application-related functionality.

* **Enable Applications Module**\
  When disabled, all application features are hidden from players and moderators.
{% endstep %}

{% step %}
## Tickets Module

Controls all ticket-related functionality.

* **Enable Tickets Module**\
  When disabled, all ticket features are hidden from players and moderators.
{% endstep %}
{% endstepper %}
