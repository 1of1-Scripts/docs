# App Customization

{% stepper %}
{% step %}
### App Customization

Customize your apps appearance through color and zoom settings.

Enter the **App Customization** menu via the dashboard.

<figure><img src="../../../../.gitbook/assets/image (91).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
### Color Customization

Customize the primary color for the entire app (landing page/dashboard)

Option A) use the text box and input a color code if you community uses a specific/unique primary color.

<figure><img src="../../../../.gitbook/assets/image (88).png" alt=""><figcaption></figcaption></figure>

Option B) click on the color sample and drag the slider on the color scale to choose a color, or use the sample tool to select a color from your display (dropper icon), finetune/tweak the color using the RGB units (optional).

<figure><img src="../../../../.gitbook/assets/image (86).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../../../.gitbook/assets/image (87).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
### Zoom Customization

Want the fonts and images to be larger or smaller across the board? Use the zoom scale to adjust the scale of the interface.

Use the minus (-) and plus (+) buttons to adjust to your preference.

<figure><img src="../../../../.gitbook/assets/image (89).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
### Save Changes

Remember to save your changes for them to take affect!

<figure><img src="../../../../.gitbook/assets/image (90).png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}
