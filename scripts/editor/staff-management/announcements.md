# Announcements

{% stepper %}
{% step %}
### Announcement Menu

On your side-bar in the Web Queue Dashboard, under the **Staff Management** section enter the **Announcements** menu.

<figure><img src="../../../.gitbook/assets/image (27).png" alt="" width="215"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
### Selecting Announcement Type

{% hint style="warning" %}
**Announcement Limits**

You can create up to 5 announcements per type. Drag to reorder them. Only active announcements will be displayed.
{% endhint %}

**You can choose between either:**

A) **Banner** Announcements\
OR\
B) **Sidebar** Announcements

Do so by either: \
\- Selecting the category (see below) then clicking "**Add First Announcement**" within that section, or,\
\- Click "**Add Announcement**" in the top right corner.

<figure><img src="../../../.gitbook/assets/image (32).png" alt="" width="563"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
### Creating an Announcement

Click "**Add Announcement**"

<figure><img src="../../../.gitbook/assets/image (33).png" alt=""><figcaption></figcaption></figure>

Select the **Type** (Banner vs. Sidebar) using the drop-down menu.

<figure><img src="../../../.gitbook/assets/image (34).png" alt="" width="340"><figcaption></figcaption></figure>

Add a **Title** (what is the announcement about?)

<figure><img src="../../../.gitbook/assets/image (35).png" alt="" width="340"><figcaption></figcaption></figure>

Then add a **Description** (more details/info for all staff to know)

<figure><img src="../../../.gitbook/assets/image (36).png" alt=""><figcaption></figcaption></figure>

Select its **Status**, if the "**Active**" status is enabled then the announcement will be displayed.

You can create multiple announcements and change their Status to Active or Inactive, you can create a catalogue of announcements to enable/disable as needed.&#x20;

<figure><img src="../../../.gitbook/assets/image (37).png" alt=""><figcaption></figcaption></figure>

Click "**Create**" to add the announcement. If **Active** status is **enabled**, it will **automatically display**, if it is disabled when you click "**Create**" you can **Edit** the announcement to **enable** the **Active** status or change information in the announcement, **Update** the change and it will be displayed.

<figure><img src="../../../.gitbook/assets/image (41).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../../.gitbook/assets/image (40).png" alt="" width="320"><figcaption></figcaption></figure>

You can drag and drop the announcements in the list to organize them as necessary!&#x20;

<figure><img src="../../../.gitbook/assets/image (38).png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
### Banner Announcement Display

In a staffs Web Queue Dashboard, the **Banner** announcement will display at the **top of their page**.&#x20;

They can **cycle** between the **Active** announcements and click into the announcement to view additional details.

<figure><img src="../../../.gitbook/assets/image (47).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../../.gitbook/assets/image (48).png" alt="" width="437"><figcaption></figcaption></figure>
{% endstep %}

{% step %}
### Sidebar Announcement Display

If a sidebar announcement is created, it will be display in the bottom of the sidebar with the option to view a specific announcement if multiple are active.

<figure><img src="../../../.gitbook/assets/image (46).png" alt=""><figcaption></figcaption></figure>

{% endstep %}
{% endstepper %}
