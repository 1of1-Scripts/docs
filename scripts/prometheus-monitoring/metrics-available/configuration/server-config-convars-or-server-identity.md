# Server Config Convars | Server Identity

### Server Config Convars

| Metric                                    | Description                            |
| ----------------------------------------- | -------------------------------------- |
| `scripts_1of1_sv_max_clients`             | Max client slots                       |
| `scripts_1of1_sv_game_build`              | Enforced game build number             |
| `scripts_1of1_sv_pure_level`              | Pure level (0=off, 1=enforced)         |
| `scripts_1of1_sv_phone_explosions`        | Networked phone explosions enabled     |
| `scripts_1of1_sv_onesync_mode`            | OneSync mode (labeled)                 |
| `scripts_1of1_sv_hostname_info`           | Server hostname (labeled)              |
| `scripts_1of1_sv_license_key_set`         | Whether license key is set             |
| `scripts_1of1_sv_endpoint_privacy`        | Endpoint privacy enabled               |
| `scripts_1of1_sv_scripthook_allowed`      | ScriptHook allowed                     |
| `scripts_1of1_sv_project_info`            | Project name (labeled)                 |
| `scripts_1of1_total_resources`            | Total registered resources             |
| `scripts_1of1_resources_by_state`         | Resources by state (labeled)           |
| `scripts_1of1_registered_commands`        | Total registered commands              |
| `scripts_1of1_sv_request_paranoia`        | Request paranoia level (0-3)           |
| `scripts_1of1_sv_networked_entity_states` | Networked script entity states enabled |
