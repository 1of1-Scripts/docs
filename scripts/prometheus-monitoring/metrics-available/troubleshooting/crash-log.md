# Crash Log

### Crash Log

| Metric                         | Description                                           |
| ------------------------------ | ----------------------------------------------------- |
| `scripts_1of1_crash_log_entry` | Individual crash entry (ts, player, crash\_id, x/y/z) |
| `scripts_1of1_crash_geo`       | Crash location with fake lat/lon for map              |
| `scripts_1of1_crash_log_count` | Total entries in crash log                            |
| `scripts_1of1_crash_latest_ts` | Unix timestamp of most recent crash                   |
