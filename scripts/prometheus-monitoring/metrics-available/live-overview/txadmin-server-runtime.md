# txAdmin - Server Runtime

### txAdmin - Server Runtime

| Metric                                      | Description                             |
| ------------------------------------------- | --------------------------------------- |
| `scripts_1of1_tx_server_boot_count`         | Total boot events                       |
| `scripts_1of1_tx_server_boot_duration_last` | Last boot duration (s)                  |
| `scripts_1of1_tx_server_close_count`        | Total close events                      |
| `scripts_1of1_tx_server_close_reason`       | Last close reason (labeled)             |
| `scripts_1of1_tx_snapshot_count`            | Total data snapshots                    |
| `scripts_1of1_tx_fxs_memory_mb`             | FXServer memory (MB)                    |
| `scripts_1of1_tx_node_memory_mb`            | Node.js memory (MB)                     |
| `scripts_1of1_tx_player_count`              | Player count (txAdmin view)             |
| `scripts_1of1_tx_svmain_ticks_*`            | svMain thread tick buckets/sum/total    |
| `scripts_1of1_tx_svnetwork_ticks_*`         | svNetwork thread tick buckets/sum/total |
| `scripts_1of1_tx_svsync_ticks_*`            | svSync thread tick buckets/sum/total    |
