# Cloudflare Direct Join

### What is this used for?&#x20;

When hosting a FiveM server, players typically connect using an IP address and port (e.g., `123.45.67.89:30120`). While this works, it's not very user-friendly or professional. This setup guide helps you:

* **Link your FiveM server to a custom domain** like `play.yourdomain.com`
* **Easily manage DNS records** through a clean Cloudflare dashboard
* **Prepare for advanced services** like DDoS protection or tunnel-based reverse proxy (e.g., Cloudflared)

This is especially useful for branded communities or hosting providers who want players to join via a clean and memorable URL.

> Example: Instead of typing `123.45.67.89:30120`, players can join `play.yourdomain.com`
