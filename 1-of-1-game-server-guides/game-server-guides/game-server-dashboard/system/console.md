# Console

### Console Terminal

The Console Terminal will be your favourite, most handy tool, it serves to control your server in real-time by:

* Inputting commands
* Power options for your server (Start, Restart, Stop, and Kill)

<figure><img src="../../../../.gitbook/assets/image (182).png" alt=""><figcaption><p>Console Terminal</p></figcaption></figure>

### Resource Usage

Displays your CPU usage and Memory usage on two separate live graphs. This can help identify issues with your server as it is running, with time stamps.

<figure><img src="../../../../.gitbook/assets/image (183).png" alt=""><figcaption><p>Resource Usage Graphs</p></figcaption></figure>

A simple, yet powerful, tab will important tools for your server's success and smooth operation.
