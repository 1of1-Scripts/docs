# SFTP Details

{% hint style="danger" %}
We strongly recommend not messing with this feature unless you are an experienced user or have created a backup of your current server. Managing files presents risks like file or data corruption, loss, or irreparable damage
{% endhint %}

The Secure File Transfer Protocol is a tool that provides a secure connection to your game server. It allows administrators to securely access, manage, and transfer files from a local computer to the server.&#x20;

Use the account details provided to use SFTP through a service like FileZilla.

The account details are:

* Connection Address, click "**Launch SFTP**" to begin.
* Username: \[your account's email].\[UUIDI]
* Password: The same password you used to log in to the Game Server Dashboard.
