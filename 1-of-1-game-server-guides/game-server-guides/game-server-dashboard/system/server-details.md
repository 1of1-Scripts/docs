# Server Details

Server Name

You can enter a new name for your server, simply type in the **Server Name** text box and click "**Update Server Name**".

* This name only displays in your Game Server Dashboard server list, it does not change the hostname.

<figure><img src="../../../../.gitbook/assets/image (184).png" alt=""><figcaption><p>Server Name in Server Details Tab</p></figcaption></figure>

### Server Limits & Info

This panel provides a broad overview of your servers:

* Name, total CPU, Memory, & Disk allowance.
* Hostname, Version #, Map, Player Slots/Online Player Count, IP Address and Port, & UUID.

<figure><img src="../../../../.gitbook/assets/image (185).png" alt=""><figcaption><p>Server Limits and Info Panel</p></figcaption></figure>

### Resource Usage

The same Resource Usage graphs from the Console tab will appear here too!&#x20;
