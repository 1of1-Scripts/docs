# Advanced

The Advanced tab displays:

### Toggle Support OP Access

* Enabling this setting will allow your support OPs to access the server's dashboard until it is disabled.

### Health Monitor

* Monitors your server and automatically restarts it if server functions are unresponsive for more than 15 seconds.
  * Some games do not support this, and this function will be null.

### Install Different Editions

{% hint style="warning" %}
Do not change this unless you have powered off your server!
{% endhint %}

{% hint style="info" %}
This example will use a Minecraft Server. However, the features, functions, and instructions remain the same for other games.
{% endhint %}

To change your Game Servers Game Edition and/or Version:

* Use the dropdown under **Edition** and select your desired option, and;
* Use the dropdown under **Version** and select the version you would like, and;
* It is **highly recommended** that you **enable the Format function**, click the **toggle button** to enable it.
* Click "**Install Different Edition**"
* Wait for this process to complete then you are free to startup your new server!
