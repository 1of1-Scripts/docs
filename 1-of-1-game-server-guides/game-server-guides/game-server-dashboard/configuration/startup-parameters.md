# Startup Parameters

This tab will display all the items your server will use to deploy itself:

#### Startup Command

The startup command is automatically injected into the console when you click "**Start**", the command will change based on the **Server File** and **Server Version,** the server boot will pull from these two items by changing the startup command based on their contents.

<figure><img src="../../../../.gitbook/assets/image (202).png" alt="" width="563"><figcaption><p>Startup Command</p></figcaption></figure>

Items that can be changed within this tab will vary by game server, however, the most common settings that can be changed here are:

* Server name, password, and auto-updates.

{% hint style="warning" %}
If you are looking to change these settings, ensure that you **power off your server** before changing anything!
{% endhint %}
