# Modpack Manager

The Modpack Manager tab will be your best friend for installing your favorite modpacks onto your server. It's as easy as clicking "Install" then "Play"!

{% hint style="info" %}
This example will use a Minecraft Server. However, the features, functions, and instructions remain the same for other games.
{% endhint %}

To install a modpack:

* Start by powering off your server in the **Console** tab by clicking "**Stop**", and;

<figure><img src="../../../../.gitbook/assets/image (214).png" alt=""><figcaption><p>Powerdown Your Server</p></figcaption></figure>

* Go to the **Modpack Manager** tab in the Tools menu, and;
* Find the modpack you like, the tab displays:
  * The modpacks name
  * Description of the modpack
  * Total number of downloads
  * and the "**Install**" button

<figure><img src="../../../../.gitbook/assets/image (215).png" alt="" width="563"><figcaption><p>Modpack Manager Overview</p></figcaption></figure>

* Click "**Install**", and;
* A prompt will appear after clicking "**Install**", make sure you:
  * Select the Version you would like to install by using the drop-down, and;
  * **Enable the Format option** so that all files will be compatible, switch the **toggle button to on**, and;

<figure><img src="../../../../.gitbook/assets/image (216).png" alt="" width="471"><figcaption><p>Installation Prompt</p></figcaption></figure>

* Go back to the **Console** tab and click "**Start**".

<figure><img src="../../../../.gitbook/assets/image (217).png" alt="" width="563"><figcaption><p>Start Up Your Server</p></figcaption></figure>

Congratulations!
