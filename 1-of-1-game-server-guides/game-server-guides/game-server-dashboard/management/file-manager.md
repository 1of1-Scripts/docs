# File Manager

{% hint style="danger" %}
This library will allow you to manage the core files of your minecraft server. **Do not** do so unless you have already created a **backup of your current server** (learn how to do so in the [Backups guide](https://app.gitbook.com/o/5YLMlSyyQwxiSl3dRjE9/s/lHfZFuj7SFT1ogoFLluQ/~/changes/3/minecraft/minecraft-game-server-dashboard/management/backups)). Do not add/remove contents in this library unless it is **through an SFTP** and you are an **experienced user**.

When modifying files within this library, it is always best to **power off your serve**r to ensure there are no data hiccups, damage, or oopsies. **Always protect your precious data!**
{% endhint %}

{% hint style="info" %}
Basic modifications are okay, such as the boolean settings within the server.properties file.
{% endhint %}
