# IP Management

{% hint style="warning" %}
This Dashboard is only available to those selected to have access, please do not request access as this is currently a Beta testing environment.
{% endhint %}

### Locating your IP(s)

Log into the DDoS Dashboard, you will be greeted with the home page.

On the left-hand side, there is a drop-down called "**Overall Management**", within the drop-down select "**Manage IPs**". (see below)

<figure><img src="../../../.gitbook/assets/image (33).png" alt=""><figcaption><p>Manage IPs menu</p></figcaption></figure>

Within the All IPs page, you will see a list contain each of your assigned IPs that are connected to your servers.

The following options will be present beside each IP:

* View Attacks
  * Simply click the eye icon to view the attacks log on the IP.
* View Rules
  * Click the cog wheel icon to view/create/manage the last synced rules on your IP.
* View Webhooks
  * Click the link icon to create/manage webhooks.
