# Creating Rules

{% hint style="warning" %}
This Dashboard is only available to those selected to have access, please do not request access as this is currently a Beta testing environment. Thank you for your understanding!
{% endhint %}

On the "Manage IPs" page, click the cogwheel icon on the IP that you're wanting to manage.

Sometimes IPs will have filters that are already present, but they may not appear automatically. Or the filters have been changed by another user and you don't have the most recent version on your Shield.

{% hint style="success" %}
**To verify that the IP's filters are the most up-to-date, be sure to click the "Sync" button at the top-right each time!**
{% endhint %}

### **This guide will cover the following:**

* Batch Games Rules for FiveM Citys (Game Filters)
* Adding Singular Rules (Network Filters)
* Creating Profiles of Preset Rules
* Multiple ports using same rule sets
