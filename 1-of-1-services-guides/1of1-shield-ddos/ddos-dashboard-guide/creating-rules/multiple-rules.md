# Multiple Rules

## Adding Multiple Rules

Click the "**Add New..**." drop-down menu, then click "**Create Game Filters**".

<figure><img src="../../../../.gitbook/assets/image (35).png" alt=""><figcaption><p>Create Game Filters</p></figcaption></figure>

Once inside the new page to create game filters, if you have multiple servers running on one machine, simply add each game server port to the appropriate rule (TCP/UDP) with a comma in-between each, see below.

<figure><img src="../../../../.gitbook/assets/image (262).png" alt=""><figcaption><p>Multiple ports with same rules set</p></figcaption></figure>
