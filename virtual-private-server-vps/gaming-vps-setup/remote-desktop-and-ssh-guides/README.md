# Remote Desktop & SSH Guides

The way you log in to your VPS depends on your choice of operating system, we developed two brief tutorials for both options:

1. Remote Desktop is for those who elected to install Windows on their VPS
2. Secure Shell Keys (SSH Key) are for those who elected to install either Debian or Ubuntu on their VPS
