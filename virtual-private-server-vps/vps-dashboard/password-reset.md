# Password Reset

Steps to Reset Your Password

{% hint style="danger" %}
We have created a YouTube tutorial for this process below:
{% endhint %}

{% embed url="https://youtu.be/iDroeNR3-5k" %}

Navigate to your [VPS Dashboard](https://dashboard.1of1servers.com) and click Manage on the Server whose password you are changing.

<figure><img src="../../.gitbook/assets/image (85).png" alt=""><figcaption></figcaption></figure>

This will bring you to your Server's Dashboard, locate the Options tab then navigate down to the Root/Administrator Password section and click Reset Password.

<figure><img src="../../.gitbook/assets/Reset Passw.png" alt=""><figcaption><p>Reset Password Location</p></figcaption></figure>

### User Selection

{% hint style="success" %}
Select "Administrator" if you installed Windows OS.
{% endhint %}

{% hint style="info" %}
Select "root" if you installed Linux on your VPS
{% endhint %}

Click Reset and the new password will be delivered to the email linked with your 1OF1 Servers account.

<figure><img src="../../.gitbook/assets/Reset Pw User.png" alt=""><figcaption><p>Reset Password User Selection</p></figcaption></figure>

{% hint style="warning" %}
Please allow 2-3 minutes to receive the email before reattempting the password reset.

Check ALL folders, spam, junk, promotions for the email.&#x20;
{% endhint %}
