# Network

### The Network tab displays:

#### Primary Network

* Displays your network; ID, MAC address, Interface, Speeds (inbound/outbound), and Type (Private/Public)

<figure><img src="../../.gitbook/assets/IPv4 info (2).png" alt=""><figcaption><p>IPv4 Network Information</p></figcaption></figure>

#### IPv4 Addresses

* Displays your IPv4; Address, RDNS, Gateway, Netmask, and Resolvers
  * If you forgot your login or did not receive your login credentials email, you can locate you IPv4 address here.

<figure><img src="../../.gitbook/assets/IPV4 address info.png" alt=""><figcaption><p>IPv4 Addresses</p></figcaption></figure>

#### IPv4 Public DNS Resolvers

{% hint style="danger" %}
Please **do not change these settings** unless you are an **experienced or advanced user** or are otherwise **told to do so by a staff of 1of Servers.**

**You can do more harm than good**
{% endhint %}

* A server requires two DNS resolvers. You may swap the system defaults for a preferred set of public resolvers under the "Primary" and "Secondary" dropdowns.
